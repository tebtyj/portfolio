Ward Management

Summary Concept:

A program that allows hospital staffs to view and manage patients, incuding visits. Some enhancements I did to this program is to ensure users would input the correct date format and if a patient had already been admitted they would not be able to admit again causing duplicates.


Features:

Able to admit,discharge patients.

Able to register,end visit.


Application used:

Eclipse (Java)
